EDA Report: Indonesian Food Image Dataset

This report documents the exploratory analysis design for the image dataset used in Forum 04. It is aligned with the training notebook and focuses on dataset structure, label balance, image geometry, and qualitative sample inspection.

This workspace currently does not contain Kaggle credentials or local image files, so this HTML report is provided as a structured report scaffold. Execute EDA_Report_Indonesian_Food.ipynb after configuring Kaggle access to populate the analysis with real charts and dataset statistics.

Dataset Scope

The project uses the Kaggle dataset rizkyyk/dataset-food-classification. The classification task covers 13 Indonesian food categories and compares a custom CNN against a VGG16 transfer learning model.

Task
Multiclass image classification
Input
RGB food images
Output
Food category label
Target Size
224 x 224 x 3

EDA Objectives

  • Measure the number of images per split and per class.
  • Inspect whether the dataset is balanced across the 13 food categories.
  • Understand image width, height, aspect ratio, and file size variation.
  • Verify sample images visually to detect mislabeling or obvious noise.

Key Analytical Sections

  • Dataset inventory: total image count, total class count, available splits, and file extensions.
  • Label distribution: bar plots for image counts per split and per food category.
  • Geometry analysis: histograms of width, height, aspect ratio, and file size.
  • Sample gallery: one representative image per category for qualitative inspection.
  • Data quality notes: interpretation of imbalance, image-size variance, and preprocessing implications.

Expected Findings

The most important expectation from this EDA is confirmation that raw images are not geometrically uniform. That result justifies resizing and center cropping before CNN training. The second major expectation is to detect whether category counts are reasonably balanced; if not, training can become biased toward majority classes.

A third expected insight is whether some food classes are visually similar enough to make top-3 accuracy materially more informative than strict top-1 accuracy.
