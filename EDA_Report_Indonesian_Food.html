EDA Report: Indonesian Food Image Dataset

Laporan ini merangkum exploratory data analysis (EDA) untuk dataset gambar makanan Indonesia yang digunakan pada Forum 04. Isi laporan diselaraskan dengan notebook utama dan menyoroti struktur dataset, keseimbangan label, variasi geometri gambar, serta implikasinya terhadap preprocessing dan pelatihan model.

Versi HTML ini merupakan ringkasan statis dari analisis yang telah disiapkan pada EDA_Report_Indonesian_Food.ipynb. Jika notebook dijalankan ulang dan divisualisasikan kembali, file HTML ini sebaiknya diekspor ulang agar tetap sinkron dengan keluaran notebook terbaru.

Cakupan Dataset

Proyek ini menggunakan dataset Kaggle rizkyyk/dataset-food-classification. Tugas yang dikerjakan adalah klasifikasi multikelas untuk 13 kategori makanan Indonesia, kemudian membandingkan performa CNN kustom dengan model transfer learning berbasis VGG16.

Tujuan EDA

  • Mengukur jumlah gambar per split dan per kelas.
  • Menilai apakah distribusi data relatif seimbang pada 13 kategori makanan.
  • Memahami variasi lebar, tinggi, rasio aspek, dan ukuran file gambar.
  • Memeriksa sampel gambar secara visual untuk mendeteksi noise atau potensi salah label.

Temuan Utama

  • Inventaris dataset: dataset lokal proyek berisi 6.490 gambar yang telah tersimpan di dalam folder dataset/.
  • Komposisi split: data terbagi menjadi 5.193 gambar train, 650 gambar validation, dan 647 gambar test.
  • Keseimbangan label: distribusi kelas relatif seimbang, terutama pada data validation dan test yang mendekati 50 gambar per kelas.
  • Variasi geometri: ukuran gambar tidak seragam, dengan rentang dimensi yang sangat lebar, sehingga standardisasi input menjadi kebutuhan penting.
  • Implikasi preprocessing: temuan ini mendukung penggunaan resize dan center crop ke ukuran 224 x 224 sebelum pelatihan CNN.

Pada data train, sebagian besar kelas berada di kisaran 398 sampai 400 gambar. Pada validation, setiap kelas berisi 50 gambar. Pada test, hampir seluruh kelas berisi 50 gambar, dengan sedikit variasi pada beberapa kategori. Distribusi ini menunjukkan bahwa dataset cukup baik untuk eksperimen klasifikasi multikelas tanpa ketimpangan kelas yang ekstrem.

Implikasi untuk Model

Hasil EDA memperlihatkan bahwa citra mentah tidak memiliki ukuran yang seragam. Variasi lebar dan tinggi gambar sangat besar, sehingga model CNN tidak dapat langsung menerima seluruh gambar tanpa tahap standardisasi. Oleh karena itu, pipeline preprocessing pada notebook utama menggunakan resize pada sisi terpendek, diikuti center crop ke ukuran 224 x 224 x 3.

Selain itu, karena distribusi kelas relatif seimbang, perbedaan performa model pada eksperimen lebih mungkin dipengaruhi oleh kualitas representasi fitur dan kapasitas model daripada bias distribusi label. Hal ini menjadi dasar yang baik untuk membandingkan CNN kustom dan VGG16 secara lebih adil.

Catatan Penggunaan

Untuk menyegarkan isi laporan ini berdasarkan notebook EDA yang paling baru, lakukan langkah berikut:

  • Pastikan folder dataset/ sudah tersedia atau Kaggle credential sudah dikonfigurasi.
  • Jalankan EDA_Report_Indonesian_Food.ipynb dari awal hingga akhir.
  • Ekspor notebook yang telah dieksekusi ke HTML dari VS Code atau Jupyter bila diperlukan.
